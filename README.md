# THE FRONT ROW - Next.js Website

A premium Next.js + Tailwind website for THE FRONT ROW event.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js App Router pages and layouts
- `/components` - Reusable React components
- `/public` - Static assets (fonts, videos, images)

## Key Features

- Left vertical navigation with blur effects
- Fullscreen hero video with interactive spotlight
- Klaviyo signup overlay
- 3D logo animation
- Scroll reveal animations
- Accessibility support (prefers-reduced-motion)

## Assets Needed

Place these files in `/public`:
- `La Foonte Sans.otf` - Brand font
- `Noa 3-D logo.mp4` - 3D rotating logo video
- `hero-video.mp4` - Hero background video (or update path in Hero.tsx)
